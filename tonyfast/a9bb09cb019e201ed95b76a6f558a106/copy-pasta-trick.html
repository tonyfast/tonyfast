shhh! it’s a secret! tell your friends¶

there is a under utilized feature in jupyter, it might as well be an easter🐰🐤egg.

if you bear with me we’ll strike gold.

reducing notebooks to code objects or even literary objects does them a disservice. they are multilingual, multimedia canvases for collage. image can be a powerful force in a narrative, but we become burdened with content management resposibilities. this mag🪄c i’m about to show will take away all those aches and pains.

this punchline is we can paste images directly into markdown cells. 🤯 right? yea, i know. its wild. so what is happening?

  1. in a markdown cell only

  1. paste an image CTRL+V

  2. jupyter saves the image data in the notebook data itself as an attachment.

see it has been in the nbformat schema the whole time, but nigh in the docs.

  1. you, the hero of this story, will recieve some text back in the form of a markdown image syntax. it starts with an exclamation because it’s :smiley_cat: for you. you’ll see sumpin like "![image.png](attachment:6c412849-cb07-4b64-90cc-a46f56c63dbc.png)"

conclusion¶

👏👏👏 we are now enlighted. we know how to make collections of images in notebooks with very natural copy paste operations. try it out, it feels good.

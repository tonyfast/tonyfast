# cleaner integration of `pidgy` & `voila`

`pidgy` works great with widgets, so it should rock with `voila`.
the default display for pidgy does NOT use widgets by default, the author would have to opt into widgets in development.
however, if we detect that `voila` is driving our notebook then we should use the `pidgy.displays.IPyWidgetsHtml` display.
